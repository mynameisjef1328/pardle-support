Quick fixes

  • Force close the app, reopen, and try again.
  • Confirm you have an internet connection (Wi-Fi or cellular).
  • If you joined a shared game, make sure everyone is using the same game code.
  • If something looks stuck, refresh or re-open the game to resync.

If you are seeing a specific error message, send it to support along with what you tapped right before it happened.

How Pardle works (in plain English)

Pardle is a golf scorecard with a Wordle-like twist. Each hole is a tiny puzzle of scoring. Enter scores, track totals relative to par, and share a game with friends so everyone stays in sync.

  • Totals are shown relative to par (even is 0, over par is +, under par is -).
  • Multiplayer games sync live when everyone is online.
  • Notes per hole are optional, but great for bragging rights.

FAQs

My game is not syncing with my friends. What do I do?
First, confirm everyone is in the same game using the same game code. Next, make sure everyone has a connection. If it still does not update, force close the app and reopen. If the issue continues, contact support and include the game code.
I refreshed and my scores changed. Why?
In a shared game, the most recent saved values win. If two people are editing the same hole at the same time, the last saved update can overwrite earlier edits. Best move is to coordinate hole-by-hole in multiplayer.
How is the total calculated?
Pardle calculates each hole as (score minus par). Then it sums across holes. Example: a 5 on a par 4 is +1. A 3 on a par 4 is -1. Even par stays 0.
Do I need an account?
No. Pardle is designed to be quick. Some versions may use anonymous authentication in the background to support syncing, but you do not create a username or password.
I purchased a course pack, but I do not see it.
Try restoring purchases (if that option exists in your build), then restart the app. If it still does not show, contact support and include the Apple ID receipt email or transaction date. Do not send your full Apple ID password.
